## Использование
При скачанном плагине, в Visual Studio Code нажимаем комбинацию клавишь ctrl+shift+P, выбираем "менеджер тем" и в открывшемся окне выбираем нужные настройки окружения. Чтобы узнать текущие настройки нажимаем "Загрузить Текущие". Чтобы отменить последнее изменение нажимаем "Отменить изменения".
